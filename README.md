## Getting Started

### Installing

* Clone this project.
    ```
    git clone https://github.com/Adirsimona/image-cropper-demo.git
    ```
Then:
    ```
    cd image-cropper-demo
    ```
Then run:
    ```
    npm i
    ```

### Executing program

```
ng serve
```
